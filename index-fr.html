API & Serveur Minecraft Multi-coeur

Photon est le serveur de nouvelle génération. Il vous offre la performance et l'extensibilité dont vous avez besoin.

Qu'est-ce que Photon

Le projet Photon comporte deux parties : une API pour les développeurs de plugins, et une implémentation du serveur, qui permet aux gens de jouer ensemble. Son but est de résoudre les problèmes des logiciels actuels (serveur vanilla, Bukkit, etc.) qui sont basés sur du code propriétaire offrant de faibles performances "single-thread".

Nous travaillons sur des solutions intelligentes pour créer la meilleure expérience pour les joueurs, les admins et les développeurs.

Pourquoi Photon est génial

Photon est une approche moderne du serveur minecraft.

Indépendant & Ouvert

Photon est un projet open-source, créé à partir de rien. Il est indépendant du code propriétaide de mojang.

Multi-thread

Votre rêve devient réalité : un serveur minecraft vraiment multi-thread, qui exécute les tâches en parallèle pour gérer efficacement plus de joueurs !

Plus rapide que la lumière

Optimisé à sa base, il vise à offrir de bonnes performances : TPS élevés et utilisation de RAM réduite.

Super API

L'API Photon est à la fois simple et puissante. Elle permet aux développeurs de créer de fantastiques plugins. Ils ont même accès aux paquets !

Actualités

Voici les évènements majeurs qui sont arrivés dernièrement

17/07/2016
Plein de choses se passent en ce moment: API des entités, implémentation du nouveau modèle de threading et des deux nouveaux types de plugins : WorldPlugin et ServerPlugin, chargement des mondes depuis le disque, ...

24/06/2016
L'authentification des joueurs et le chiffrement des paquets sont désormais fonctionnels !

22/05/2016
TheElectronWill a proposé un nouveau modèle de "threading", plus simple et plus efficace. Lire le sujet.

16/05/2016
Les paquets ont été mis à jour pour Minecraft v1.9.4

13/04/2016
L'API v0.3.0 (alpha) est disponible ! Elle apporte de nombreuses améliorations.

04/04/2016
Photon utilise désormais le système de build Gradle. Lisez le README.md du dépôt github pour savoir comment l'utiliser.

26/03/2016
L'API Photon est désormais disponible en ligne, sur github.
