What is Photon and why do we need it?

Photon is a minecraft server software. It's a program that allows people to play together. There are already two big server softwares out there: the mojang's one ("vanilla server"), and the bukkit's one ("craftbukkit"). But they have several problems:

  1. CraftBukkit is built on top of the mojang's vanilla server, so it depends on mojang's program which is very difficult to understand.
  2. CraftBukkit is facing legal issues, therefore you can't download it as usual.
  3. They are not using all the available computing power we have today, resulting in poor performance sometimes.
  4. They do not allow devs to modify the core of the game.
The goal of photon is to build an alternative software, free of all these issues. Photon needs to be fast, extensible and independant from the vanilla server. There are also newer community projects for minecraft, like the Sponge API. But Sponge doesn't aim to create an independant multi-core server.

The benefits of Photon

Independant

Photon is made from scratch and with open libraries. It is therefore free from mojang's proprietary code. Moreover, a large part of its code will be available on github.

Faster than light

Optimized at its core, it aims to deliver lightning-fast performace: high TPS and low RAM usage.

Multi-Threaded

You've asked it for years, it's becoming true: a 100% multi-threaded server that uses all the available power of the modern computers.

Extensible

Designed to be extensible, Photon allows the devs to modify every single aspect of the game. Plugins can listen to events involving blocks, entities and players. They can add new blocks, items, entities, world generators and permissions system. They even have access to packets! The only limit is your imagination.
